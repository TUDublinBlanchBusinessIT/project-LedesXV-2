# **Meal Tracker**

This project is a simple PHP and MySQL web application. It allows users to record their daily meals and store them in a database. The system will display all saved meals.

## **What the Project Does**
- Records meal information  
- Stores data in a MySQL database  
- Displays saved meals on screen  
- Tracks calories, protein, fat and carbs  
- Records the date of each meal 
- Records the type of meal (breakfast, lunch, dinner, snack)
